# Stack Overflow Survey Analysis (Educational Project)

## Project Overview
This project is based on the Stack Overflow Developer Survey dataset.  
The goal was to explore developer trends, analyze compensation patterns, and investigate Python popularity across different groups.

This is an educational project created to practice data analysis using real-world data.

## Objectives
- Clean and preprocess survey data
- Perform exploratory data analysis (EDA)
- Analyze Python popularity by age group
- Calculate percentage distributions
- Identify high-income remote workers (75th percentile)
- Explore industries among top-earning professionals

## Tools & Technologies
- Python
- Pandas
- Jupyter Notebook

## Key Analysis Performed
- Data cleaning and handling missing values
- GroupBy analysis
- Percentage calculations
- Descriptive statistics
- Percentile-based filtering
- Compensation analysis
- Industry frequency analysis

## Key Insights
- Python popularity varies significantly across age groups.
- Remote high-income professionals are concentrated in specific industries.
- Compensation distribution shows strong upper percentile differences.

## Project Structure
- `stack_overflow_analysis.ipynb` — main analysis notebook

## Dataset
Public dataset from Stack Overflow Developer Survey.
